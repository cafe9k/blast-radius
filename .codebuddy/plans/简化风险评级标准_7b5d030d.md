## 产品概述

简化 Blast Radius 工具的风险评级标准，使其更加直观易懂

## 核心功能

- **风险评分简化**：仅使用代码行数 (lineCount) 作为风险分数，行数越多表示耦合大且难维护
- **节点大小调整**：以代码行数相对值确定节点大小，行数越多节点越大
- **风险等级重定义**：
- Low: 0-100 行
- Medium: 101-300 行
- High: 301-500 行
- Critical: 500+ 行

## 技术栈

- 后端：TypeScript + Node.js
- 前端：React + TypeScript + Tailwind CSS
- 图可视化：Graphology + Sigma.js
